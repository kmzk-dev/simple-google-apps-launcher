# Simple Google App Launcher

ChromeのツールバーからGoogleアプリランチャーを自分好みに。
アイコンのスタイルやサイズを自在にカスタマイズでき、ドラッグ&ドロップで簡単に並び替えが可能です。

<img src="/ss/ss_01.png">

## 概要
この拡張機能は、ブラウザのポップアップから Gmail、Drive、Gemini などの Google アプリを即座に起動できるようにします。ユーザーは自分の好みに合わせてアイコンの並び順を変更したり、頻繁に使うアプリだけを表示するようにフィルタリングしたりすることが可能です。

## 特徴

- **クイックアクセス**: ツールバーのアイコンから主要なGoogleサービスへ即座にアクセス。
- **自由な並び替え**: ドラッグ&ドロップでアイコンを好きな順番に入れ替え可能。
- **スタイルカスタマイズ**: 標準（Original）とドット（Dot）の2種類のアイコンテーマを選択可能。
- **サイズ調整**: 利用シーンに合わせてアイコンサイズを「小（Small）」と「大（Large）」から選択可能。
- **表示管理**: オプション画面から、使用しないアプリを非表示に設定可能。

## プロジェクト構造
```
chrome-launcher/
├── dev/
│   ├── manifest.json         # 拡張機能の設定ファイル (Manifest V3)
│   ├── default_apps.json     # アプリケーションのマスターデータ
│   ├── popup.html / .js      # ランチャーのメイン画面とロジック
│   ├── options.html / .js    # 表示設定画面
│   └── statics/              # アイコン・画像リソース
```

## クイックスタート
本プロジェクトをローカル環境で Chrome に読み込む手順です。

1. 本リポジトリをクローンまたはダウンロードします。
2. Google Chrome を開き、アドレスバーに `chrome://extensions/` を入力して実行します。
3. 右上の 「デベロッパー モード」 を ON にします。
4. 「パッケージ化されていない拡張機能を読み込む」 をクリックします。
5. ダウンロードしたプロジェクト内の `dev` ディレクトリを選択します。

## 使い方

### ランチャーの起動
- アイコンから: ブラウザ右上の拡張機能アイコンをクリックしてポップアップを開きます。
- ショートカットから: キーボードの `Alt` + `L` を押すと、瞬時にランチャーが起動します。

### アプリの起動と並べ替え
- アイコンをクリックすると、該当する Google サービスが新しいタブで開きます。
- アイコンを長押ししてドラッグすることで、好きな位置に並べ替えることができます。
- ポップアップ下部の 「Default Order」 ボタンをクリックすると、アイコンの順序を `default_apps.json` の定義順に戻せます。

### 表示設定の変更
1. 拡張機能アイコンを右クリックし、「オプション」 を選択します。
2. **アイコンのスタイル**: 「Default」または「Dot Style」を選択して保存します。
3. **アイコンのサイズ**: 「Default」または「Large」を選択します。選択に合わせてランチャーの幅も自動調整されます。
4. **アプリの表示/非表示**: チェックボックスを切り替えることで、ポップアップに表示するアプリを選択できます。

## 開発について
### アプリケーションの追加
`default_apps.json` に新しい項目を追加し、`statics/icons/` に該当するアイコンを配置することで、ランチャーに新しいサービスを追加できます。

### 技術スタック
Vanilla JavaScript, CSS, HTML5, Chrome Extension API (V3).
